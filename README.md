# 오승준 포트폴리오

전자공학과 졸업생 오승준의 개인 포트폴리오 사이트입니다.
GitHub Pages를 통해 호스팅되며, 순수 HTML/CSS/JS로 제작되었습니다.

**Live:** https://5tmdwns.github.io

---

## 구성

| 섹션 | 내용 |
|---|---|
| Home | 인트로 |
| About | 취미, 특기, 프로젝트 수 |
| Education | 학력 타임라인 |
| Skills | 사용 언어, IDE, 툴 아이콘 목록 |
| Projects | 프로젝트 타임라인 (GitHub 링크 포함) |
| Patent | 출원 특허 |
| Contact | 주소, 이메일 |

## 로컬 실행

```bash
git clone https://github.com/5tmdwns/5tmdwns.github.io.git
cd 5tmdwns.github.io
# index.html을 브라우저에서 바로 열거나 Live Server 등으로 실행
```
