## Overview ⭐️

- Github Pages를 활용한 정적 웹 포트폴리오
- HTML / CSS / Bootstrap 기반
- AOS(Animate On Scroll) 라이브러리를 활용한 스크롤 애니메이션
- 페이지 전환 애니메이션 및 반응형 레이아웃 지원

## Structure 📁

```
5tmdwns.github.io/
├── index.html              # 메인(홈) 페이지
├── about.html              # 소개 페이지 (Skills / Education / Interest)
├── portfolio.html          # 작업물 목록 페이지
├── portfolio-details.html  # 작업물 상세 페이지
├── contact.html            # 연락처 페이지
├── css/
│   ├── custom.css          # 커스텀 스타일
│   ├── bootstrap.min.css
│   └── aos.css
├── js/
│   ├── jquery.min.js
│   └── aos.js
└── images/                 # 프로필 사진, 파비콘, 스킬 아이콘 등
    └── icons/
```

## Pages 📋

- **Home** — 히어로 섹션 (소개 문구 및 프로필 사진, 인쇄 최적화 포함)
- **About** — 기술 스택(Skills) / 학력(Education) / 관심사(Interest) 탭
- **Works** — 프로젝트 카드 그리드 (최신순, 9개씩 페이지네이션)
- **Works Detail** — 각 프로젝트 상세 설명
- **Contact** — 위치 및 이메일 연락처
